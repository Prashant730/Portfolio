# How to Save Your Project Screenshots

## Step-by-Step Instructions:

1. **Save the screenshots you shared as these exact filenames in the `/public` folder:**
   - **Learning Tutor** (first image) → `learning-tutor-preview.jpg`
   - **Career Guidance** (second image) → `career-guidance-preview.jpg`
   - **BidMaster** (third image) → `bidmaster-preview.jpg`
   - **BirdNet** (fourth image) → `birdnet-preview.jpg`

2. **How to save them:**
   - Right-click each image you shared
   - Select "Save image as..."
   - Navigate to your project's `/public` folder
   - Use the exact filename from above
   - Choose JPG format

3. **File locations should be:**
   ```
   public/
   ├── learning-tutor-preview.jpg
   ├── career-guidance-preview.jpg
   ├── bidmaster-preview.jpg
   └── birdnet-preview.jpg
   ```

## ✅ Project data is already updated!

The code now expects these exact filenames and will automatically use them as backgrounds.

## 🎯 Result:

Once you save the images, each project card will show your actual project screenshot as the background with a dark overlay and project info on top.
